1. Introduction

Now a days we see that a lot of gun violence incidents in schools, colleges, restaurants & in public places. A lot of things are responsible for these incidents such as racial discrimination, mental health, bullying etc. In order to reduce mass shootings, the policy makers & the public at large need to know the various reasons for occurance of these events, the shooter's motives etc, and these informations can be obtained through data classification & analysis. And hence the motivation for our project.

2. Tasks and Approach

Analysing the Gun Violence data and classifying the data set depending on the various metrics after applying classification models.
● We shall set some hypothesis to be drawn from our dataset after the classification models are applied on the dataset.
● We shall clean and preprocess the dataset.
● We shall also figure out the important fields in the dataset for analysis and answering some hypothesis.
● Developing and applying the algorithms and bringing out the results in a tabular format or graphs. We are considering Naive Bayes, Logistic Regression as a classifier and KNN classifier algorithm for now. The technologies we will be using are Java, Python,Apache Spark. Logistic regression will be applied to hypothesis which has binary classification like.

3. Data set

The dataset we have chosen is Gun violence data with records dated from 2013 to 2018. The source of the dataset is Kaggle. The entire data set has 260k records and 29 features. The data set is of 34MB size. Out of the 29 features, we will only be using certain features for our analysis. In order to obtain the required data we will have to do preprocessing and combine data from multiple features. The following features are the once which we will be using :
- State (indicates state of the incident): used to classify the region of crime (West, Midwest, Northeast, Southwest, Southeast, Pacific)
- Participant_age_group, participant_type: used to classify the age group of the suspect
- Participant_type, participant_status: used to classify status of suspect after the incident
- Data: used to classify gun violence based on year
- n_killed , n_injured: Used to classify the severity of the incidence
- Participant_gender, Participant_type: used to classify the gender of the suspect.
